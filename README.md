# oculante
A no-nonsense image viewer

I started this as a toy project to make a simple image viewer. Here are the features:

Image format support:
- bmp	
- dxt	
- gif	
- hdr	
- ico	
- jpeg	
- png	
- pnm	
- tga	
- tiff	
- webp	

Platform support:
- Linux
- Mac
- Windows

Threaded image loading



Planned:
- Display unpremultiplied images correctly
